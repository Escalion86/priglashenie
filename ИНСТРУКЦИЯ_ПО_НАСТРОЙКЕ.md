## Что нужно сделать пользователю:

### 1. Создание Google Таблицы

1. Перейдите по адресу https://sheets.google.com
2. Создайте новую таблицу
3. Назовите её, например, "Ответы гостей"
4. Скопируйте ID таблицы из URL (часть между `/d/` и `/edit`)
   - Пример: `https://docs.google.com/spreadsheets/d/ABC123XYZ/edit` → ID = `ABC123XYZ`

### 2. Настройка Google Apps Script

1. В открытой таблице перейдите в меню **Расширения** → **Apps Script**
2. Удалите код, который там есть по умолчанию
3. Скопируйте код из файла `google_apps_script.js` (этот проект)
4. **Важно**: замените строку `const SHEET_ID = 'ВАШ_SHEET_ID_ЗДЕСЬ'` на ваш реальный ID таблицы
5. Нажмите иконку дискеты (Сохранить) или Ctrl+S

### 3. Развертывание скрипта

1. Нажмите кнопку **Развертывание** (справа вверху)
2. Выберите **Новое развертывание**
3. В поле "Тип" выберите **Веб-приложение**
4. Настройки доступа:
   - **Выполняется от имени**: Ваш аккаунт
   - **Доступ**: "Доступно всем, в том числе анонимным пользователям" ⚠️ (это важно!)
5. Нажмите **Развернуть**
6. Скопируйте появившийся **URL веб-приложения** (он начинается с `https://script.google.com/...`)

### 4. Обновление кода сайта

1. Откройте файл `index.html`
2. Найдите строку: `const scriptURL = 'YOUR_SCRIPT_URL_HERE'`
3. Замените `YOUR_SCRIPT_URL_HERE` на скопированный URL из шага 3.6

### 5. Тестирование

1. Откройте сайт в браузере
2. Заполните форму внизу страницы
3. Нажмите "Отправить"
4. Проверьте, появились ли данные в Google Таблице

## Структура данных в таблице:

После первой отправки в таблице автоматически создадутся заголовки:

- Дата отправки
- Имя гостя
- Присутствие (yes/no/later)
- Диета
- Примечание к диете
- Напитки
- Спутник (alone/withPartner)
- Имя спутника
- Комментарий

## Возможные проблемы:

- **Ошибка доступа**: Убедитесь, что при развертывании выбрали "Доступно всем, в том числе анонимным пользователям"
- **Данные не появляются**: Проверьте правильность ID таблицы и URL скрипта
- **CORS ошибки**: В коде уже добавлен режим `no-cors`, это нормально
